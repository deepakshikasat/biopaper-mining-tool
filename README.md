# BioPaper Mining Tool

    LLM-ready literature mining app for extracting structured data from PubMed abstracts.

    ## Dataset

    Public PubMed/E-utilities workflow with small included demo extraction table.

    ## Methods

    - NCBI E-utilities search adapter
- JSON-style extraction schema
- Streamlit results browser

    ## Reproduce

    ```bash
    python scripts/run_pipeline.py
    ```

    ## Outputs

    - `outputs/extracted_abstracts.csv`
- `figures/modality_counts.svg`
- `src/pubmed.py`
- `app.py`

    ## Portfolio Note

    This repository uses public or synthetic demonstration data only. It is
    intended to show reproducible computational biology and AI/ML workflow
    design without relying on confidential or employer-owned material.
