BioPaper Mining Tool

Bench to Bytes portfolio analysis report. Public data sources are named explicitly.

Hypothesis

A public PubMed query can be converted into a structured table for triage of drug-induced liver injury mechanisms.

Data

NCBI PubMed E-utilities query for 'drug induced liver injury mechanism', retrieving up to 30 public abstracts.

Process

  1. Fetched PubMed IDs with ESearch.
  2. Fetched abstracts with EFetch XML.
  3. Applied transparent rule-based extraction for compounds/topics, mechanism keywords, and toxicity mentions.
  4. Generated mechanism count outputs and a Streamlit browser.

Outputs

  • outputs/extracted_abstracts.csv
  • outputs/mechanism_keyword_counts.csv

Interpretation

This is now a real public literature workflow. Rule-based extraction is less powerful than an LLM, but it is reproducible, auditable, and provides a baseline for future LLM extraction.
